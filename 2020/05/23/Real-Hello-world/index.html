Real Hello world

这应该是实际意义上第一篇真正的Blog了

为什么决定要自己建Blog

原因无他，就是晚上有个无聊的讲座，可能有些重要的内容不得不听，但是又不想全程听，就搭建一下 Blog 来打发时间（当然不是大家都有 Blog 然后我这个自己在本地文件夹里面默默写 markdown 的酸了）

其实主要原因还是很多新学习的一些小知识点，小技巧很容易在记忆中迅速遗漏，所以很有必要多多做些记录。特别是一些环境搭建类的，很多自己搭建时候费九牛二虎之力才跳出来的坑，帮别人搭建的时候又遇到了，但是忘了解决方案，就只能放弃作罢（特别是还是自己提出帮助别人的）

以后更新 Blog 的计划

先会找一个时间集中上传一下在别处的 Blog 和自己之前写的一些笔记，以后尽量做到学到新东西就做记录，既方便自己复习整理，也方便某一方面的学习在被更重要的 DDL 打断之后可以很方便的断点恢复

关于这个 Blog

现在只做了个基本功能，完全可以称其为毛坯房，以后有时间慢慢研究 hexo 的主题什么的，还有社交媒体，其他一些链接什么的。寒假租过一个阿里云的小容器，最终计划肯定是把自己的 blog 完全部署在那上面，但是现在，还是有很长的路要走呀。
